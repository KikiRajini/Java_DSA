When Two Pointers is the Right Choice
1) Array is sorted
   Classic signal.
   Examples
   • Two Sum II
   • 3Sum
   • Remove duplicates from sorted array
   • Container with most water
2) You need to find pairs / triplets
   Examples
   • Pair with target sum
   • Count pairs
   • 3Sum / 4Sum
3) You compare from both ends
   Examples
   • Palindrome check
   • Reverse string
   • Valid palindrome
4) In-place modification required
   Examples
   • Move zeros
   • Remove element
   • Partition array
   • Sort colors
5) Problems asking for:
   • Minimum length
   • Maximum distance
   • Closest sum

For char array:
